
# Creation Steps #
1. Create html directory, and put static files in there
2. Create server.js file
3. npm init
4. npm install express mysql
5. Create a file called .env, with an entry:
    DB_PWD=[your-db-password]
    DB_NAME=[db-name]

Run normally:
node server.js
